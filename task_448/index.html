В коде используется функция Number для преобразования значения атрибута value в числовой формат. Это необходимо, чтобы избежать конкатенации строк и гарантировать, что происходит числовое сложение с 1. Без использования Number может возникнуть конкатенация строк, что может привести к неправильным результатам.
