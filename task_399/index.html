Приведенный код использует событие "click" на кнопке. Когда происходит клик по кнопке, запускается функция обработчика события. Внутри функции происходит инверсия текущего состояния атрибута disabled у элемента elem с помощью логического NOT (!). Если elem.disabled равен true, то после инверсии он станет false, и наоборот. Таким образом, состояние disabled инвертируется при каждом клике на кнопку.
